PoB Trade Helper — Settings

Starting the server

uv run python3 api/run_mcp.py \
  --host 0.0.0.0 \
  --port 8080 \
  --api-secret YOUR_TOKEN

The server must be reachable from your browser. If it's on a remote machine, make sure it's behind HTTPS (nginx + Let's Encrypt, Caddy, or Tailscale HTTPS).
